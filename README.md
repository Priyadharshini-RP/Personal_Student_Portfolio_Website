# Personal_Student_Portfolio_Website

A modern and responsive personal portfolio website developed using **HTML, CSS, and JavaScript** to showcase my skills, education, projects, certifications, achievements, and resume. The website features a clean user interface with smooth animations, dark/light mode support, typing animation, responsive design, and interactive navigation.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Dark and light mode toggle
* Animated typing effect on the home page
* Interactive navigation with active section highlighting
* Skills section with animated progress bars
* Education, experience, and project showcase
* Certifications and achievements sections
* Resume view and download option
* Contact form with client-side validation
* Smooth scroll animations using Intersection Observer
* Dynamic footer displaying the current year

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome Icons
* Google Fonts (Poppins)

## Project Structure

* `index.html` – Website structure
* `style.css` – Styling and responsive design
* `script.js` – Interactive functionality and animations
* `resume.pdf` – Resume file
* `image.jpg.png` – Profile image

## Purpose

This portfolio serves as a personal website to present my technical skills, academic background, projects, and achievements while demonstrating modern front-end web development practices.
